## Installation

1. Clone this repository.
2. Create a `.env` file with the following variables:
	```env
	DB_HOST=
	DB_USER=
	DB_PASSWORD=""
	DB_NAME=

	PAYSTACK_PUBLIC_KEY=
	PAYSTACK_SECRET_KEY=
	```
3. Open your terminal and run:
	```bash
	composer install
	```
4. Create the database.
